3. ### JsCssZip

   1. 是一款压缩js和css的小工具，可以减轻代码的重量，减少服务器压力，能将网页代码中的js和css代码进行压缩、精简，去除冗余代码，并保持代码原有功能。压缩的原理主要是将空格和注解去掉！还有就是将变量或语法简化。但是在不改变原始JS的作用前提下
